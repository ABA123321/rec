Rune Abyss — brand logo pack (SVG)
==================================
与站点组件 RuneAbyssLogo 一致：金质圆徽、四向刻度、符文 A、青绿内环与底点。

文件说明
--------
logo-mark-full-color.svg      主图标 256×256 视口，全渐变，通用导出
logo-mark-solid-gold.svg      实金底 + 深色线，适合 favicon / 小尺寸
logo-mark-mono-dark.svg       深色线稿，适合浅色背景、印刷线稿
logo-mark-mono-light.svg      浅色线稿，适合深色海报 / 视频角标
logo-lockup-horizontal.svg    横排：图标 + Rune Abyss + 中文副标（深色场景）
logo-lockup-horizontal-light-bg.svg  同上，适合白底 / PPT
logo-lockup-stacked.svg       竖排：图标在上 + 字标（深色场景）
